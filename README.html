Local Setup

Clone git clone https://github.com/<your_username>/portfolio-website.git && cd portfolio-website
Install cd backend && npm install && cd ../frontend && npm install
Run backend cd backend && npm run dev → http://localhost:4000
Run frontend cd frontend && npm run dev → http://localhost:3000
Verify Visit http://localhost:3000 and test the API at http://localhost:4000/api/projects

Deployment

Production builds run npm run build in /frontend and /backend. Lint the UI with npm run lint in /frontend and confirm both dev servers start cleanly before deploying to Vercel/Render.

Consult AGENTS.md for coding style, testing guidelines, and PR formatting standards.

Full-stack portfolio built with Next.js 14, Express, and TypeScript. Deployed via Vercel (frontend) and Render (backend).
